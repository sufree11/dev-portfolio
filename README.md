# dev-portfolio
Portfolio for all projects and education achievements.

## Custom Fonts

This UI expects the following font hierarchy:

- SKIP STD B: menu and body text
- RODIN PRO: headings
- NEW RODIN PRO: large stylized text
- BM SPACE: logo text

For deterministic rendering across machines, place your licensed webfont files in:

- public/fonts/

Expected filenames are documented in public/fonts/README.md.
