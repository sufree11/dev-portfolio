# Fonts Setup

Place your licensed font files in this folder using the exact filenames below.

- skip-std-b.woff2 (or skip-std-b.woff)
- rodin-pro.woff2 (or rodin-pro.woff)
- new-rodin-pro.woff2 (or new-rodin-pro.woff)
- bm-space.woff2 (or bm-space.woff)

Notes:
- The app will use these files first for consistent rendering.
- If a file is missing, local installed fonts are used as fallback.
- Keep only fonts you are licensed to use.
